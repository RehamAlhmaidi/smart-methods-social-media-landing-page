# Smart Methods Social Media Landing Page

A responsive and modern landing page developed for **Smart Methods** during **Engineering Day**. The page serves as a centralized hub, allowing visitors to quickly access the company's official website, social media platforms, and location through a clean and intuitive interface.

---

## Project Overview

This project was designed to provide a simple yet engaging user experience for visitors looking to connect with Smart Methods online. The interface follows modern web design principles, featuring a responsive layout, interactive elements, and a lightweight implementation using only HTML and CSS.

---

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern glassmorphism-inspired user interface
- Direct access to the Smart Methods official website
- Quick links to LinkedIn, X (Twitter), YouTube, Instagram, and Google Maps
- Interactive buttons with hover effects
- SEO-friendly meta tags
- Open Graph meta tags for better social media sharing
- Optimized image loading using lazy loading
- Lightweight static webpage with fast loading performance

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Web Design
- Google Fonts (Poppins)
